# 🎨 SystemControl UI Design Guide\n\n本指南介紹使用 Magic MCP 和 Figma MCP 設計的 SystemControl 用戶界面組件。\n\n## 🏗️ 架構概覽\n\n### UI 組件結構\n```\nsystemControl/ui/\n├── camera_control_dashboard.tsx    # 主控制面板\n├── camera_settings_panel.tsx       # 相機設定界面\n├── dual_dial_controller.tsx        # 雙轉盤控制器\n├── film_simulation_selector.tsx    # 軟片模擬選擇器\n├── index.tsx                       # 組件索引和路由\n└── UI_DESIGN_GUIDE.md              # 本指南\n```\n\n## 🎯 設計理念\n\n### 復古相機美學\n- **色彩方案**: 深色主題配橘色重點色 (#ea580c)\n- **材質質感**: 金屬質感、類比錶盤、專業相機風格\n- **交互設計**: 觸覺反饋、平滑動畫、直觀操作\n\n### 專業攝影工作流\n- **雙轉盤系統**: 模擬專業相機的左右轉盤操作\n- **即時反饋**: 所有設定變更立即可見\n- **快捷鍵支援**: A/D (左轉盤), ←/→ (右轉盤), Space (按壓)\n\n## 📱 UI 組件詳解\n\n### 1. CameraControlDashboard\n**主控制面板** - 系統核心界面\n\n#### 功能特色:\n- 📊 即時相機設定顯示 (快門、ISO、EV、白平衡)\n- 🔄 雙轉盤控制介面\n- 🎞️ 軟片模擬選擇器\n- 🔋 電池和儲存狀態指示器\n- ⚡ 功能開關 (閃光燈、HDR)\n- 📏 復古式曝光指示錶\n\n#### 設計亮點:\n- **漸層背景**: 深灰到黑色的專業感背景\n- **卡片式佈局**: 圓角卡片增加現代感\n- **動態指示器**: 曝光值的漸變色彩顯示\n- **觸覺回饋**: 按鈕點擊和轉盤旋轉動畫\n\n#### 使用方式:\n```tsx\nimport { CameraControlDashboard } from './ui';\n\n<CameraControlDashboard />\n```\n\n### 2. CameraSettingsPanel\n**相機設定界面** - 詳細配置面板\n\n#### 功能特色:\n- 🗂️ 分頁式設定介面 (Image/Video/Focus/Exposure/Color)\n- 🎚️ 專業滑桿控制\n- 🔘 切換開關\n- 📋 下拉選單\n- 👁️ 即時預覽模式\n- 💾 設定匯入/匯出\n\n#### 設計亮點:\n- **標籤導航**: 清晰的功能分類\n- **即時預覽**: 右側預覽面板顯示設定效果\n- **自訂滑桿**: 橘色滑桿配合整體色彩方案\n- **快速操作**: 重置、儲存等常用功能按鈕\n\n#### 使用方式:\n```tsx\nimport { CameraSettingsPanel } from './ui';\n\n<CameraSettingsPanel />\n```\n\n### 3. DualDialController\n**雙轉盤控制器** - 專業相機操作界面\n\n#### 功能特色:\n- 🎛️ 實體轉盤模擬 (左轉盤模式選擇，右轉盤數值調整)\n- 🎯 8種相機模式 (快門、ISO、EV、白平衡等)\n- 🎮 滑鼠拖拽和鍵盤控制\n- ⚡ 觸覺反饋動畫\n- 📱 響應式設計\n- 💫 金屬質感和專業紋理\n\n#### 設計亮點:\n- **3D轉盤效果**: 圓錐漸變和紋理線條\n- **環形指示器**: 模式指示器圍繞轉盤分佈\n- **動態範圍顯示**: 右轉盤顯示當前模式的可調整範圍\n- **專業配色**: 左轉盤橘色，右轉盤藍色\n\n#### 使用方式:\n```tsx\nimport { DualDialController } from './ui';\n\n<DualDialController />\n```\n\n#### 操作說明:\n- **滑鼠操作**: 點擊拖拽轉盤旋轉\n- **鍵盤快捷鍵**:\n  - `A/D`: 左轉盤逆時針/順時針\n  - `←/→`: 右轉盤逆時針/順時針\n  - `Space`: 轉盤按壓功能\n\n### 4. FilmSimulationSelector\n**軟片模擬選擇器** - 藝術化軟片效果界面\n\n#### 功能特色:\n- 🎞️ 12種 Fujifilm 風格軟片預設\n- 🎨 軟片膠卷視覺效果\n- 📊 自訂參數微調 (飽和度、對比度、顆粒感等)\n- 👁️ 即時效果預覽\n- 💾 自訂預設儲存\n- 🔄 參數重置功能\n\n#### 設計亮點:\n- **膠卷美學**: 仿真軟片膠卷的視覺設計\n- **溫暖色調**: 琥珀色背景營造復古攝影氛圍\n- **網格佈局**: 響應式網格展示所有軟片類型\n- **漸變卡片**: 每種軟片使用獨特的色彩漸變\n- **動畫效果**: 選擇軟片時的縮放和發光效果\n\n#### 軟片類型:\n1. **Standard** - 標準平衡色彩\n2. **Vivid** - 鮮豔高飽和度\n3. **Provia** - 自然風景色調\n4. **Velvia** - 濃郁對比強烈\n5. **Astia** - 柔和人像色調\n6. **Classic Chrome** - 復古街拍風格\n7. **Pro Neg. Hi/Std** - 專業負片效果\n8. **Classic Neg** - 經典負片\n9. **Eterna** - 電影感色調\n10. **Acros** - 高質感黑白\n11. **Monochrome** - 純淨單色\n\n#### 使用方式:\n```tsx\nimport { FilmSimulationSelector } from './ui';\n\n<FilmSimulationSelector />\n```\n\n## 🎨 設計系統\n\n### 色彩規範\n```css\n/* 主要色彩 */\n--primary-orange: #ea580c;\n--primary-blue: #3b82f6;\n--accent-red: #ef4444;\n--accent-green: #10b981;\n\n/*   className=\"px-4 py-2 rounded-lg bg-gray-700 text-white\"\n          >\n            Dials\n          </button>\n          <button \n            onClick={() => setActiveView('films')}\n
